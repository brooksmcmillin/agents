# Task Notifier Crontab Examples
# Add these to your crontab with: crontab -e

# === Three times daily (9 AM, 2 PM, 6 PM) - Weekdays only ===
0 9 * * 1-5 cd /home/brooks/build/agents && uv run python -m agents.notifier.main >> /tmp/task-notifier.log 2>&1
0 14 * * 1-5 cd /home/brooks/build/agents && uv run python -m agents.notifier.main >> /tmp/task-notifier.log 2>&1
0 18 * * 1-5 cd /home/brooks/build/agents && uv run python -m agents.notifier.main >> /tmp/task-notifier.log 2>&1

# === Once daily at 9 AM ===
# 0 9 * * * cd /home/brooks/build/agents && uv run python -m agents.notifier.main >> /tmp/task-notifier.log 2>&1

# === Twice daily (morning and afternoon) ===
# 0 9,15 * * * cd /home/brooks/build/agents && uv run python -m agents.notifier.main >> /tmp/task-notifier.log 2>&1

# === Every 4 hours during work day ===
# 0 9,13,17,21 * * * cd /home/brooks/build/agents && uv run python -m agents.notifier.main >> /tmp/task-notifier.log 2>&1

# === Monday morning only (weekly reminder) ===
# 0 9 * * 1 cd /home/brooks/build/agents && uv run python -m agents.notifier.main >> /tmp/task-notifier.log 2>&1

# === End of day reminder (5 PM) ===
# 0 17 * * * cd /home/brooks/build/agents && uv run python -m agents.notifier.main >> /tmp/task-notifier.log 2>&1

# Notes:
# - Replace /home/brooks/build/agents with your actual project path
# - Logs are written to /tmp/task-notifier.log
# - First run will trigger OAuth (open browser) - run manually first
# - Cron format: minute hour day month weekday
# - 1-5 = Monday-Friday, * = every
